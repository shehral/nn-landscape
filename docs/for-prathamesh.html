  • GRPO fine-tuning supply chain for IDP Leaderboard-comparable models is now complete in public tooling: ms-swift (training on Qwen3-VL, InternVL3.5, GLM4.5v), LLaVA-OneVision-1.5 (fine-tune base), and evalscope (evaluation) trended together for the third consecutive build cycle.

  • Ollama's featured model list has become a real-time proxy for which open-weight VLMs are production-ready for local deployment: Kimi-K2.5, GLM-5, gpt-oss, Qwen, and DeepSeek are now one-command installs, directly enabling on-premise document AI benchmarking without managed-API cost.

  • Five agent orchestration platforms (Dify, RAGFlow, LangChain, UltraRAG, Genkit) trended this cycle; none has announced a documented Nanonets OCR-3 integration, representing a persistent distribution gap that recurs across build cycles without resolution.
