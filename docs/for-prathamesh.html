Patterns this build

  • Open-source orchestration convergence: Dify, RAGFlow, BiSheng, Flowise, and AnythingLLM all trended in a single cycle, presenting a complete document automation stack without API spend — each sets document extraction defaults at adoption time.
  • Dual evaluation harness consolidation: lmms-eval and evalscope both trended with overlapping DocVQA/ChartQA/OmniDocBench coverage — competitors can run structured head-to-head comparisons in either harness without OCR-3 as a registered baseline.
  • Fine-tuning toolchain plus local distribution convergence: ms-swift GRPO support for Qwen3-VL and GLM-5.1 appeared alongside Ollama's expanded roster of those same models, pairing an accessible training path with a zero-install local inference channel.
  • Extraction-surface distribution gap in production AI frameworks: genkit, langchain, dify, and liteparse all appeared in the same cycle; genkit and langchain serve frontier-lab APIs natively, while dify and liteparse bundle document extraction defaults that displace Nanonets at adoption.
  • VLM audit tooling is maturing alongside evaluation harnesses: crossroute-audit (explanation-faithfulness) trended in the same cycle as lmms-eval and ms-swift — a research community now has open-source tools to characterize, evaluate at scale, and fine-tune against VLM hallucinations.
