Patterns this build

  • Document-to-structured-output pipelines proliferating as open-source tools: MinerU, RAGFlow, and LlamaIndex all converge on 'PDF/Office to LLM-ready markdown/JSON' as their primary positioning — the same use case as Nanonets Agentic Data Extraction. Open-source availability anchors the price floor for the category.
  • LlamaIndex repositions from orchestration framework to 'leading document agent and OCR platform': this is a deliberate competitive encroachment, not an adjacency. LlamaParse (ParseBench, Parse-Flow, Agentic Plus bounding boxes) and LiteParse form an expanding extraction product suite under an OCR-primary brand.
  • Local VLM inference infrastructure (Ollama, vLLM-Omni, TokenSpeed) expanding support toward OCR-adjacent model families: GLM-5.1 (related to Zhipu AI's GLM-OCR) and DeepSeek (DeepSeek-OCR 2) now first-class in Ollama's headline. Reflects growing demand for self-hosted document-AI inference and a cost-avoidance path for smaller customers.
  • VLM faithfulness auditing emerging as a distinct tooling category: CrossRoute-Audit (VLM explanation-faithfulness) and video-evaluator (grounded evidence extraction) both trending on GitHub in the same window, signaling that explainability tooling for multimodal models is moving from research artifact to software product.
  • Tesseract continues to trend as a top OCR repo: sustained developer interest in the legacy baseline confirms that the OCR replacement market remains active and developers are actively benchmarking. Models that publish Tesseract-relative accuracy gains have a ready audience.
