Patterns this build

  • The full open-source document AI stack — extraction (MinerU, Unstructured), orchestration (Dify, BiSheng, JeecgBoot), and model training (LLaVA-OneVision-1.5, TinyLLaVA_Factory) — is trending in a single build cycle, covering every pipeline layer without a proprietary component.
  • MCP is consolidating as the default API integration protocol: UltraRAG and JeecgBoot both adopt MCP-native connectors this cycle, continuing a multi-build trend where every new orchestration platform ships MCP support as a first-class feature.
  • Chinese-backed open-source orchestration platforms (BiSheng, Dify, JeecgBoot, UltraRAG) and US-backed enterprise frameworks (Genkit, LangChain) are trending simultaneously, presenting two distinct distribution channel gaps for any extraction service without connectors in either ecosystem.
  • Two accessible VLM training frameworks (LLaVA-OneVision-1.5 and TinyLLaVA_Factory) trended in the same cycle alongside NVIDIA NeMo Automodel, indicating a near-term wave of low-cost VLM fine-tunes that could generate IDP Leaderboard submissions without proprietary training infrastructure.
  • Retrieval-based hallucination mitigation (self-reflective multimodal RAG) continues to appear without an explicit comparison to mechanistic interpretability approaches; this is the fourth or more consecutive build in which a retrieval-based VLM hallucination method has trended.
