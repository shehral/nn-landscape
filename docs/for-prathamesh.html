Patterns this build

  • Open-source PDF-to-structured-data tooling is proliferating: PaddleOCR, opendataloader-pdf, and Unstructured.io all trended this cycle with identical document-parsing positioning, widening the zero-cost open-source alternative to managed extraction APIs.
  • Document VLM evaluation harnesses are consolidating around two independent frameworks: lmms-eval and evalscope both trended in the same cycle with overlapping DocVQA, ChartQA, and OmniDocBench coverage, standardizing the infrastructure through which third parties publish competitor benchmarks.
  • IDP-Leaderboard-comparable VLMs are becoming one-click locally runnable: Ollama now lists GLM-5.1 as a featured model, ms-swift enables DPO/GRPO fine-tuning of Qwen3-VL and InternVL3.5, and Nexa SDK adds day-0 mobile NPU support — collectively compressing the cost and friction of deploying competitive document VLMs.
  • Image compression as a customer cost-reduction strategy is entering the document API toolkit: latent-gate's VL-JEPA pipeline (~80% token reduction) introduces a new class of upstream middleware that modifies the payload quality reaching extraction APIs, with unknown impact on document accuracy.
  • Explanation-faithfulness auditing is emerging as a distinct VLM research methodology: crossroute-audit provides mechanistic tools for verifying whether model explanations match internal representations, complementing behavioral hallucination evaluation with interpretability-layer auditing.
