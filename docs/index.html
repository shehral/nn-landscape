Patterns this build

  • Frontier-lab releases this build land on agent-action and document-extraction surfaces, not pure capability: Gemini 3.5 with action, Gemini Omni multimodal, Antigravity 2.0, Databricks GPT-5.5 enterprise agents, and Nemotron 3 Nano Omni for documents all bundle agent execution with VLM extraction.
  • Open-weight model releases continue compressing the competitive set on the IDP Leaderboard surface: Gemma 4, DeepSeek V4, Kimi K2.6, MiMo 2.5, GLM-5.1, Qwen3.7-Max and PaddleOCR 3.5 all dropped within the build window.
  • Multimodal evaluation infrastructure is converging on unified harnesses: lmms-eval, evalscope, the Open Agent Leaderboard, and ms-swift all target one-toolkit-many-models VLM evaluation rather than per-benchmark scripts.
  • Long-context multimodal positioning is becoming explicit: Nemotron 3 Nano Omni and DeepSeek-V4 both frame million-token-class context as document-and-agent-usable, raising structural questions about chunked OCR pipelines.
  • Open-source PDF parsing tooling continues to receive momentum: opendataloader-pdf, PyMuPDF, RAGFlow, and UltraRAG all trended on GitHub. The open floor below paid OCR APIs keeps moving up.
