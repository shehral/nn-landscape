Patterns this build

  • The GRPO fine-tuning supply chain for IDP Leaderboard-comparable models is now complete in a single build cycle: ms-swift for GRPO training of Qwen3-VL, Qwen3-Omni, InternVL3.5, Ovis2.5, and GLM4.5v; XTuner for MoE-scale training; LLaVA-OneVision-1.5 for sub-7B models.   • Four orchestration-layer platforms (Dify, BiSheng, FastGPT, LangChain) trend without confirmed Nanonets Agentic Data Extraction connectors. The bundled-default-parser pattern — where orchestration platforms select parsers at deploy time — is the structural distribution risk, not direct API competition.

Combined with Unstructured.io trending as the orchestration-layer default parser, the open-source document AI pipeline is both complete and accessible without API cost.
