  • The self-hosted open-source document pipeline is complete and co-trending: Tesseract (OCR engine), OCRmyPDF (PDF OCR), opendataloader-pdf (AI-ready parser), and paperless-ngx (document archive) appear in the same build cycle — representing the full capture-to-archive workflow without API dependencies.

  • High-performance multimodal model serving is becoming commodity infrastructure: SGLang and vllm-omni both trend as dedicated serving frameworks for large VLMs, shifting competitive differentiation from serving capacity toward model accuracy and API surface quality.
  • Ollama's local model access now covers multiple IDP Leaderboard-adjacent competitive models (GLM-5.1, Kimi-K2.6, Qwen, gpt-oss) in the same featured set as the benchmark harnesses above; together, these allow a developer to benchmark OCR-3 competitors locally, without API keys, using a standardized evaluation CLI.
