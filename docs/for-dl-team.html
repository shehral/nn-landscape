Patterns this build

  • The full open-source document AI pipeline — extraction (Unstructured), orchestration (Dify, BiSheng), RAG (UltraRAG, all-in-rag) — trends in a single cycle for the third or more consecutive build. The stack now covers every tier without proprietary components.
  • VLM training framework fragmentation is widening: LLaVA-OneVision 1.5, TinyLLaVA Factory, and NVIDIA NeMo Automodel all trend in the same cycle, offering three distinct accessible paths from dataset to a deployable multimodal model. The structural barrier to producing an IDP Leaderboard entrant has declined.
  • MCP is appearing as a distinct distribution channel: UltraRAG and JeecgBoot both reference MCP-native integration in the same cycle. Any extraction service without an MCP connector is structurally absent from pipelines built to the emerging MCP standard.
  • High-performance multimodal serving (SGLang, NVIDIA NeMo) and accessible VLM training (LLaVA-OneVision 1.5, TinyLLaVA Factory) trend together, suggesting the bottleneck to fielding an OCR-3-class model is shifting from training compute to distribution channel coverage.
  • Hallucination-aware retrieval-based mitigation and small-scale VLM training frameworks converge on overlapping ablation territory: both offer paths to testing whether VLM hallucination patterns transfer across architectures at sub-OCR-3 compute cost, without requiring a full-scale interpretability experiment.
